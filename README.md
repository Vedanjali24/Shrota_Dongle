# Shrota_Dongle
The Shrota Dongle is a custom-designed Ground Control Station (GCS) telemetry interface PCB developed as part of our rocketry team’s avionics and communication system. The board enables reliable reception and processing of real-time telemetry transmitted from the rocket during flight.

The system is built around an STM32 microcontroller, which manages telemetry data handling and communication with a LoRa RYLR module used for long-range wireless data reception. This allows the ground station to receive critical flight parameters and system status during launch and testing.

The PCB features a 5.5 V to 3.3 V regulated power architecture with VCC filtering to ensure stable operation of RF and digital components. High-speed and low-speed oscillators provide precise timing, while external SPI Flash memory allows telemetry data to be logged for redundancy and post-flight analysis.

Developed collaboratively as part of our team’s ground infrastructure, the Shrota Dongle provides a robust communication link for monitoring mission-critical rocket performance.
